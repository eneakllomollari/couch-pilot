## [0.1.0] - 2024-01-03

### Added

- Initial release
- AI-powered TV control via natural language chat
- Support for Fire TV and Google TV devices via ADB
- TP-Link Tapo L530 smart bulb integration
- Virtual remote control UI with D-pad, volume, and app launcher
- Deep linking for Netflix, HBO Max, Apple TV+, YouTube
- Keyboard shortcuts for remote control
- WebSocket-based real-time chat interface
- REST API for direct device control
- Pydantic Settings for type-safe configuration
- GitHub Actions CI pipeline
- Pre-commit hooks for code quality

### Security

- Credentials moved to environment variables
- Added security documentation for local-network-only deployment
